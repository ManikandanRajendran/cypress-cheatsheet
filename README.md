# Cypress — Command Log Cheatsheet

> Cypress's API rebuilt as a working homage to its own Test Runner — a command log on the left, time-travel snapshots on the right. From your very first `npm install` to advanced sessions and custom commands.

**🌐 Live site:** [ManikandanRajendran.github.io/cypress-cheatsheet](https://ManikandanRajendran.github.io/cypress-cheatsheet)
**❓ Interview Q&A:** [.../cypress-qa.html](https://ManikandanRajendran.github.io/cypress-cheatsheet/cypress-qa.html)

## What's inside

Plus a standalone **Interview Q&A page** with 20 questions across 5 topics: fundamentals & architecture, the async command queue, network/fixtures/test data, custom commands & sessions, and real-world debugging scenarios.

## Design

The whole page is styled as a faithful homage to Cypress's own Test Runner — the real jade-green Cypress brand color, a fake browser chrome bar, and a click-to-time-travel command log that mirrors exactly how the real tool's debugging UI works.

## Features

- Command log + snapshot panel, just like the real Test Runner
- Live search across all commands
- Category filter chips
- Day/Night theme toggle, persisted via localStorage
- Standalone Q&A page with topic navigation and search
- Zero dependencies — single HTML file per page

## Local development

```bash
git clone https://github.com/ManikandanRajendran/cypress-cheatsheet.git
cd cypress-cheatsheet
open index.html
```
